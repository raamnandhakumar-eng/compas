# Deviations from preregistration

## Before the live audit

The repository originally used four occupations, one base profile per occupation, two temperatures, and 1,280 placebo evaluations. Before any live Claude result was observed, the confirmatory design was revised to:

- eight occupations;
- four base profiles per occupation;
- one locked primary temperature;
- 512 matched resumes;
- 2,560 planned evaluations;
- three primary outcomes: fit score, recommendation, and confidence;
- a separate name-source screen and perception-pretest gate.

The original 1,280-evaluation run remains a software-validation benchmark and is not treated as the confirmatory live design.

## After the live audit

No post-result deviations exist because the live Claude audit has not been run. Any later change must be dated, justified, and labeled confirmatory or exploratory before the affected analysis is reported.
