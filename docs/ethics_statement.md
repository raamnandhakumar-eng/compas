# Ethics statement

This repository uses synthetic candidates only. It must not be used to rank real applicants or make employment decisions.

Names are studied as perceived name signals. The project does not infer or assign an actual protected identity to any person. Aggregate Census and SSA statistics are used only for stimulus source screening, and a separate consented perception pretest is required before demographic interpretation.

Human participants must receive an informed-consent statement, may withdraw where required, and should not provide unnecessary sensitive personal information. Institutional review should be sought when required by the host institution.

The audit preserves null findings, negative findings, failed responses, refusals, and deviations. It does not selectively rerun observations based on their outcomes. Reported effects must be tied to the exact model, prompt, date, and synthetic sample.

The project cannot establish unlawful discrimination, intent, or real employer behavior. Any public communication should avoid claims about actual people or the economy as a whole.
